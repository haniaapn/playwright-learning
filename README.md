# Playwright Automation Framework

Enterprise-grade QA Automation Framework using Playwright + TypeScript

## Features
- TypeScript-based framework
- Page Object Model
- Typed Fixtures
- Auth Session Fixture
- Environment Config
- Parallel Execution
- CI/CD Pipeline (GitHub Actions)
- HTML Test Reporting

## Tech Stack
- Playwright
- TypeScript
- Node.js
- GitHub Actions

## Run Locally
```bash
npm install
npx playwright install
npx playwright test
